Pick an accent — this is the only saturated color in the chrome

Used for: active tab underline, selected state, focus rings, primary buttons, "live" indicators. Channel colors (speed/throttle/brake/etc.) stay unchanged.

A

Cyan — keep current vibe

Closest to today's #00d4ff speed-line. Reads "instrument", "telemetry". Pro: continuity with existing channel palette. Con: overlaps with the Speed channel color, which can confuse focus vs data.

B

Violet — premium / Linear-adjacent

Reads "modern dashboard", "software product". Doesn't collide with any data channel color. Pro: looks the most "not AI slop" out of the gate. Con: less "racing", more "SaaS".

C

Amber — motorsport-warm

Reads "pit board", "lap timing". Pro: warm, distinct from the cool data channels, very on-theme for racing. Con: warm accents on a cold UI is a strong opinion — has to be applied with restraint or it goes garish fast.
