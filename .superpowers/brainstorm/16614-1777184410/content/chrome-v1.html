Notes on what you're seeing:

  • Title bar — replaces the bare window title. Brand dot pulses violet, source pill shows the live data source + track + temps, right side shows always-on session context (lap, stint, last lap time). One row, info-dense, never moves.
  • Tabs — flat, lowercase-cap, violet underline on active. Green dot on tabs with live data flowing (Race, Strategy when alerts).
  • Cards — single border style, 8px radius, mono numbers with tabular figures (don't jitter), uppercase labels at 10px. Pills for status. Warn variant is amber-tinted, error would be red — color is reserved for state, never decoration.
